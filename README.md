# CodeReview 👨🏽‍💻
Esse foi um projeto desenvolvido como um trabalho escolar, com o objetivo de aprimorar os conhecimentos sobre Git, GitHub, Pull Request (PR) e Code Review. A entrega consiste em uma aplicação web (site), que contém informações, elaboradas pelos integrantes do grupo, sobre diferentes lugares do mundo, escolhidos com base nos gostos de cada desenvolvedor.

## Como acessar? ⚙️
É possível acessar nosso site de duas formas:
1. Clique no link presente na aba lateral da página do repositório -> https://ryancursinom.github.io/CodeReview/
2. Clona o repositório e execute localmente

## Execução local
- Primeiro, abra uma pasta no VSCode e crie um novo terminal
- Execute o comando abaixo para clonar o repositório, puxando todas as imagens, arquivos, etc
```text
git clone  https://ryancursinom.github.io/CodeReview/
```
- Execute o projeto, e acesse ele pelo navegador de sua preferência
